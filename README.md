**Sales Analysis Project**

&#x20;**Overview**



This project analyzes retail sales data using Python and Pandas.

It explores sales performance, profitability, and regional trends using real-world business data.



&#x20;**Objectives**

Analyze total sales and profit

Identify top-performing product categories

Compare sales across regions

Find best and worst performing products

Generate meaningful business insights



&#x20;**Tools Used**

Python 

Pandas 

Matplotlib 

&#x20;

**Dataset**



The dataset contains 9,994 sales transactions with columns such as:



Sales

Profit

Category

Region

Product Name

Discount

Quantity



&#x20;**Key Insights**

Technology is the most profitable category

Furniture has high sales but very low profit

West region performs best in sales

South region is the weakest market

Discounts strongly affect profitability

&#x20;

How to Run This Project

pip install pandas matplotlib

python sales\_analysis.py
